# Snippets Cheatsheet

## Meteor

* `tp` = `<template name="arg"> ... </template>`
* `ea` = `{{#each arg}} ... {{/each}}`
* `if` = `{{#if argIs 'arg2'}} ... {{/if}}`
* `Meteor.render` = `Meteor.render(function () { ... });`
* `Meteor.publish` = `Meteor.publish('name', function () { ... });`

## JavaScript

* `cl` = `console.log();`
* `fun` = function
* `if` = if statement
* `ife` = if else statement
* `ter` = tertiary conditional
* `for` = for loop
* `wh` = while loop
* `ret` = return
* `timeout` = timout function
* `switch` = switch case
* `proto` = prototype

## Ruby

* `if`
* `ife` = if else
* `for`
* `while`

### Rails

* `hm` = has many
